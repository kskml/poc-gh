You are a senior Java developer. We are validating user passwords for a web application.

TASK: Write a Java method to validate a password according to our security rules.

CONSTRAINTS:
- Rules: Minimum 8 characters, at least 1 uppercase, 1 lowercase, 1 digit, 1 special character.
- DO NOT use a single massive Regex. Use a loop with Character class methods for readability.
- DO NOT return a boolean. If validation fails, throw a custom `InvalidPasswordException` with a message listing ALL the rules the user failed to meet (e.g., "Password must contain at least one digit").
- Handle null inputs by throwing an IllegalArgumentException.

FORMAT: 
Provide the PasswordValidator class and the custom InvalidPasswordException class. Use Java 17.
